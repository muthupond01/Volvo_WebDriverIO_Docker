Pre- requistites
 
 --Node JS
 --Java script supported IDE (eg: Intellij,VSCode, Webstorm)
 --docker


 To run the tests,follow the below steps

 -- Do 'npm install' to download all the required dependencies from package.json
 -- Run the command 'docker-compose up' inorder to begin the node setup
 -- Run the command 'npm run test' to begin the execution in docker and generate reports
